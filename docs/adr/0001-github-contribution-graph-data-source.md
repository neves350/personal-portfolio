# GitHub contribution graph data source

We need to render a GitHub contribution graph on the portfolio, which is a static Angular SPA with no backend or serverless functions. GitHub's own API requires an auth token, which can't be safely embedded in client-only code, ruling it out. We chose to fetch from `github-contributions-api.jogruber.de` (a free, unauthenticated public API — the same source `react-github-calendar` relies on) and render the graph with a custom-built Angular component, rather than embedding a static image badge (e.g. `ghchart.rshah.org`). The custom component lets the graph match the site's own `--primary` color tokens and dark-mode toggle exactly, and lets each day-cell carry an accessible `aria-label` and keyboard-focusable tooltip — neither is possible with a pre-rendered image.
