# Portfolio

Martim Neves' personal portfolio site — a static Angular SPA (no backend) showcasing projects, tech stack, and work experience.

## Language

**Contribution Graph**:
The GitHub heatmap of daily activity (commits, PRs, issues) rendered as a grid of shaded squares, covering roughly the last 12 months, for a fixed GitHub username.
_Avoid_: GitHub Calendar, activity calendar, commit graph
