About

The frontend work I focus on.

Open to work · Immediately

I like building interfaces that feel right to use. Most of my work has been with Angular — components, reactive state, forms — and I care about the details that make a UI actually pleasant.

My production experience comes from a 5-month internship where I built the entire frontend of a web app for 100+ aviation students using Angular and Ionic, owning the UI/UX and integrating 5+ APIs alongside a backend team.

On personal projects I've gone fullstack — NestJS backend, Angular frontend, tested with Vitest. Enough to understand the backend side and collaborate without friction, though frontend is where I focus.

Working principles

  • Write code that's readable before it's clever.
  • Own the feature end-to-end, not just your file.
  • Design and code should talk to each other.
  • Test behavior, not implementation.
  • Keep improving until it feels right.
